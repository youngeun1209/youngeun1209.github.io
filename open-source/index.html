OpenBMI

221★ / 113 forks

Open-source toolbox and benchmarking platform for brain-computer interface development, with standardized, reproducible pipelines for MI, ERP, and SSVEP paradigms. Lead developer and technical architect.

GitHub

NeuroTalk

AAAI 2023

Generative framework for synthesizing speech from brain signals, implementing the methods from our AAAI 2023 paper on voice reconstruction during imagined speech. Includes reproducible demos.

GitHub

cIOL

IEEE TNSRE

Noise- and movement-artifact reduction method for ambulatory EEG, implementing the techniques reported in our IEEE Transactions on Neural Systems and Rehabilitation Engineering paper.

GitHub

GigaScience · ~700 citations

Standardized EEG dataset for three BCI paradigms (MI, ERP, SSVEP), released with the OpenBMI toolbox and published as a peer-reviewed data descriptor in GigaScience.
